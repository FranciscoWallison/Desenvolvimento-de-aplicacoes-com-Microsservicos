
# 📝 Gabarito Comentado: Fundamentos e Habilidades de Segurança

Este guia serve para revisão rápida dos conceitos abordados no módulo, com foco na distinção entre perfil comportamental e técnico.

### 1. Habilidades Transferíveis (Soft Skills)

**Questão:** Quais das seguintes proficiências são Habilidades transferíveis, que podem ser aplicadas em quase todos os campos? (Selecione três respostas)

* [x] **Comunicação escrita e verbal**
* [x] **Análise**
* [x] **Solução de problemas**
* [ ] Programação *(Incorreto nesta categoria)*

> **💡 Explicação da Correção:**
> O curso define **Habilidades Transferíveis** como aquelas universais, que você leva de uma profissão para outra (ex: um médico, um advogado e um analista precisam saber se comunicar e resolver problemas).
> * **Por que "Análise" está certa?** Refere-se à *capacidade analítica* ou pensamento crítico (olhar um problema e entender as partes), algo útil em qualquer emprego.
> * **Por que "Programação" está errada aqui?** É uma habilidade **técnica** específica. Nem todo profissional precisa saber programar, portanto, não é "transferível" para *todos* os campos.

### 4. Classificação de Dados (PII vs. SPII)

**Questão:** Quais são os exemplos de informações sensíveis de identificação pessoal (SPII) que os profissionais de segurança cibernética precisam proteger? (Selecione duas respostas)

* [x] **Registros médicos**
* [x] **Números de contas bancárias**
* [ ] Endereços de e-mail *(PII Padrão)*
* [ ] Sobrenomes *(PII Padrão)*

> **💡 Explicação:**
> Nem todo dado pessoal é considerado "Sensível" (SPII).
> * **PII (Personal Identifiable Information):** Dados que identificam você, mas que muitas vezes são públicos (Lista telefônica tem nome e endereço). Ex: E-mail, Sobrenome.
> * **SPII (Sensitive PII):** Dados que, se vazados, causam **dano grave** (financeiro, reputação ou segurança). Ex: Dados de saúde, financeiros, biometria e documentos oficiais (CPF/RG).
