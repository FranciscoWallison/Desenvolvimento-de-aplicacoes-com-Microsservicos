## Pré-requisitos Gerais

```
- Dispositivo rootado (físico ou emulador via RootAVD)
- ADB configurado
- Frida instalado (ver ferramentas/Frida.md)
- Burp Suite (para análise de tráfego)
- JADX (para análise estática antes dos testes)
```
