# Mobile Security

Seção dedicada à segurança de aplicações móveis, cobrindo arquitetura de plataforma, ferramentas de análise, técnicas de ataque e pesquisa em nível de kernel.

## Plataformas

| Plataforma | Foco |
|---|---|
| [Android](./Android/README.md) | Magisk, KernelSU, Frida, emulator spoofing, nível kernel |
| [iOS](./iOS/README.md) | TrollStore, Frida, bypass de jailbreak detection |

## Roadmap de Estudo

```
1. Entender a arquitetura da plataforma (camadas, sandbox, permissões)
2. Configurar ambiente (emulador rootado ou dispositivo físico)
3. Dominar ferramentas de análise estática (JADX, MobSF)
4. Dominar análise dinâmica (Frida, Objection)
5. Aprender técnicas de bypass (root detection, SSL pinning)
6. Explorar nível kernel (KernelSU, LKM, eBPF)
```
